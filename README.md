# watson-and-python
A python script that uses CHATGPT API and Google Speech Recognition.

**Notes**
  - Under development.
  - Errors may appear.
  - Python >= 3.8

**Usage**
  - Insert your openai API KEY in chatgpt.py
  - Go there and build your Assistant.
  - Run main.py.
  - Say "Alex.." to start a conversation.
